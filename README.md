# Exploratory-Data-Analysis-on-Amazon-s-Diwali-Campaign-Performance-2025

This project focuses on analyzing Amazon's Diwali Campaign Performance 2025 using Exploratory Data Analysis (EDA). The aim is to understand sales trends, profit growth, customer behavior, and product performance during the Diwali season.

The project includes sales and profit analysis, state-wise demand analysis, monthly revenue trends, top-performing products, and revenue vs profit comparison. It also uses different KPIs such as PEI, PPR, PSI, DIV, CPV, UPSI, and SMPI to generate meaningful business insights.

## Objective
- Analyze sales performance
- Understand profit and loss
- Study customer behavior
- Perform state-wise demand analysis
- Find monthly sales trends
- Create unique KPIs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Output
- Business Insights
- Customer Insights
- Product Performance Analysis
- Monthly Trend Analysis
- Final Processed CSV File
